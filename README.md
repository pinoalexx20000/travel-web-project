# Travel Web Project

Academic web project developed as part of the Web Programming subject.

This project is a travel-themed website built with HTML, CSS and JavaScript. The website includes several pages related to travel information, weather, suggestions and general content about the platform.

## Project objective

The main objective of this project was to practice the creation of a structured website using frontend technologies.

The project helped me improve my understanding of page structure, visual design, navigation between pages, JavaScript interaction and organization of static web assets.

## Features

- Multi-page website structure
- Home page
- About us page
- Travel information page
- Weather-related page
- AI suggestions page
- Form and email-related functionality
- Organized assets folder for CSS, JavaScript and images

## Project structure

- `index.html` - Main page
- `aboutUs.html` - About us page
- `clima.html` - Weather page
- `viatjes.html` - Travel page
- `sugerenciasIa.html` - Suggestions page
- `assets/CSS/` - Stylesheets
- `assets/JS/` - JavaScript files
- `assets/img/` - Images and visual resources

## Technologies

- HTML
- CSS
- JavaScript
- Responsive layout basics
- Web design
- Git/GitHub

## What I learned

This project helped me practice the basic structure of a web application, organize frontend files and improve my understanding of how HTML, CSS and JavaScript work together.

It also allowed me to work on visual design, page navigation, user interaction and basic frontend logic.
